Description

This is a project for client, who wanted to have dynamic page for his own business. I decided to created it in Wordpress, to have ability to add dynamic new lessons, events with text, photos and videos.

The child theme has:

  1. Header with Logo and menu (mobile menu)

  2. Front Page with sections:

    • Section gallery slider with description and button read more
    • Section about me
    • Section Groups
    • Section learning steps.
    • Section contact.
    • Footer
  3. Fourth Custom Archive Pages for each different group. Content is displayed for people who are logged and have added role. 
  4. Footer - User can add 4 widgets to footer.

PL

Opis

Projekt utworzony dla klienta, który chciał mieć dynamiczną stronę dla własnych celów. Stronę utworzyłam w Wordpresie, w celu zapewnienia możliwości łatwego dodawania lekcji, wydarzeń z tekstem, zdjęciami, plikami video.

Motyw potomny zawiera:

  1. Header z logo oraz menu

  2. Strona główna z sekcjami:

    • Sekcja – slider ze zdjęciami, opisem i przyciskiem więcej informacji.
    • Sekcja – o mnie.
    • Sekcja - o czterech grupach nauczania.
    • Sekcja - kroki nauczania.
    • Sekcja – kontakt.
  3. Cztery typy własnych stron (archive pages). Każda strona przeznaczona jest dla osobnej grupy docelowej. Informacje stron są wyświetlane dla osób które są zalogowane i mają określoną rolę użytkownika do wyświetlania informacji.
