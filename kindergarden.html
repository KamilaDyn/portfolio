ENG

Description

This website I created as a first website example in HTML with the Bootstrap framework. After some time when I have learnt WordPress, I decided to change HTML website into WordPress, to get more experiences. 

The website has:

  1. Front Page with sections:
    • Section with banner
    • Section with fourth cards which have links with information pages
    • Section with information.
    • Section with carousel
    • Section with four blog posts
  2. Events Page - events post type
  3. Information Pages about Kindergarten with five subpages 
  4. Blog Page 
  5. Sidebar
  6. Extra widget with events
  7. Search Input information from the website. For this, I created a WordPress route to have data in json and then write functions in jquery to get output with information. 
  8. For this website I used plugin: 
    • Advanced Custom Fields.
    • Members
    • Smush
    • Regenerate Thumbnails 
PL

Opis

Stronę Kindergarden utworzyłam najpierw z użyciem bootstrap. Po jakimś czasie, po przerobieniu kursu z Wordpress’a, zmieniłam stronę pod stronę dla cms-wordpress. W celu zdobycia doświadczenia, sprawdzenia umiejętności.

Strona posiada

  1. Stronę główną z sekcjami:

    • Sekcja z banerem.
    • Sekcja z czterema kartami informacyjnymi, każda karta prowadzi do osobnej strony.
    • Sekcja z informacją o stronie (działalności).
    • Sekcja z slajdami (wydarzenia).
    • Sekcja wiadomość – informacja o stronie (wybranie strony )
    • Sekcja z postami z bloga.
  2. Strona z wydarzeniami (utworzenie typu postów events)
  3. Strony z informacjami o działalności (5 podstron)
  4. Strona z blogiem.
  5. Sidebar
  6. Dodatkowy widget z nadchodzącymi wydarzeniami
  7. Własne pole wyszukiwania informacji na stronie. Do tego celu zstał utworzny wp route, aby pobrać json danymi do wyświetlenia.

  8. Dla strony zostały wykorzystane pluginy:

    • Advanced Custom Fields.
    • Members
    • Smush
    • Regenerate Thumbnails
