Description

This is a project for client, who wanted to have dynamic page for his own business. I decided to created it in Wordpress, to have ability to add dynamic new lessons, events with text, photos and videos.

I build own theme which contain:

  1. Header with Logo and menu (mobile menu)

  2. Front Page with sections:

    • Section gallery slider (ability to add / change 4 images).
    • Section welcome with featured picture and text.
    • Section Featured Videos (displayed two pages with video)
    • Section about (by choosing one page). Contain text and featured image.
    • Section courses (by choosing 4 pages, contained video).
    • Section holiday Programs. Section is displayed as Slider. Programs added by custom post types. Contain featured images.
  3. Fourth Custom Archive Pages for each different group. Content is displayed for people who are logged and have added role. 
  4. One Page with the calendar. In the calendar, events are automatically added from the event post type. When a user wants to add event need to write the date of the event in an advanced custom field. Also can add the colour of event, short description, tick if wants to display details as a popper in the calendar, choose priority of events (if events have the same dates). For this, I create WP json router, and used json to get data and added into js-year-calendar function. The calendar which I used it is from an open source from github. 
  5. Footer - User can add 4 widgets to footer.

  6. Also theme has: 

    • Two Sidebars
    • Extra widgets for Holiday Programs, which can be added to sidebar. Widget display just 4 earliest planned programs. 
    • Extra widgets for social links (4).

Opis

Projekt utworzony dla klienta, który chciał mieć dynamiczną stronę dla własnych celów. Stronę utworzyłam w Wordpresie, w celu zapewnienia możliwości łatwego dodawania lekcji, wydarzeń z tekstem, zdjęciami, plikami video.

Zbudowałam motyw, który zawiera.

  1. Header z logo oraz menu

  2. Strona główna z sekcjami:

    • Sekcja – slider ze zdjęciami (możliwość dodawania zmieniania 4 zdjęć).
    • Sekcja – powitalna z tekstem oraz zdjęciem
    • Sekcja about – informacja o stronie (wybranie strony )
    • Sekcja "Featured Videos", dwa posty z video.
    • Sekcja możliwe kursy (możliwość wybrania z czterech stron zawierających video).
    • Sekcja – wydarzenia. Wydarzenia są dodawane poprzez własny typ postów (event)
  3. Cztery typy własnych stron (archive pages). Każda strona przeznaczona jest dla osobnej grupy docelowej. Informacje stron są wyświetlane dla osób które są zalogowane i mają określoną rolę użytkownika do wyświetlania informacji.
  4. Strona z kalendarzem. Do kalendarza dodawane są wydarzenia z własnego typu postów, "event". Jeśli użytkownik chce dodać wydarzenie do kalendarza musi dodać datę w ACF (advance custom fields) Może również dodać opis, color wydarzenia, czy wyświetlać informacje jako popper (chmurka) na kalendarzu, oraz wybrać priorytet wydarzenia (w przypadku gdy daty się powielają). Do tego celu utworzyłam json route w celu wykorzystania json dla kalendarza js-year-calendar. Kalendarz pobrałam z open source na githubie. Kalendarz dostosowałam do potrzeb strony.
  5. Stopka, użytkownik może dodać cztery widgety do stopki.
  6. Sidebar - również strona posiada dwa sidebar’y, oraz dodatkowy widget dla wydarzeń (wyświetlanie 3 najwcześniejszych wydarzeń)
  7. Dodatkowy widget z wybraniem "social links" (4 linki z ikonami)
