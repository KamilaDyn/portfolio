Wordpress Page 'Fenix'

Website based on free Wordpress theme Rara Themes - Education Zone (link to theme)

ENG

Description

Website created with Wordpress, based on Rara Theme - Education Zone. It was a final assignment for a study 'web and mobile application'  CMS section.

Website has a customized free theme Education Zone. The website has:

  1. Comments section
  2. Sidebar
  3. Blog
  4. Four pages with information
  5. Online shop with registration, login and basket with products.

  6. Plugins

    • Contact Form 7
    • ReCaptcha v2 for Contact Form 7
    • Akismet antyspam
    • Jetpack z WordPress.com
    • Newsletter
    • WooCommerce
    • WooCommerce Admin
    • WooCommerce PayPal Checkout Gateway
    • WooCommerce Services
    • Yoast SEO
PL

Opis

Storna internetowa, stworzona z wykorzystaniem Wordpressa, motyw Rara Theme- Education Zone. Strona na potrzeby zaliczenia przedmiotu, na studia z zakresu aplikacje mobilne i internetowe dla sekcji Zarządzanie treścią (CMS).

Obecnie strona osiada dostosowany szablon Education Zone wersja darmowa. Strona posiada

  1. Dostosowanie komentarzy do strony.
  2. Sidebar
  3. Blog
  4. Cztery strony o działalności z informacjami.
  5. Dostosowanie sklepu internetowego, utworzenie nowego konta w celu zakupów, możliwość logowania do sklepu, koszyk zakupów.

  6. Na stronie zostały wykorzystane wtyczki:

    • Contact Form 7
    • ReCaptcha v2 for Contact Form 7
    • Akismet antyspam
    • Jetpack z WordPress.com
    • Newsletter
    • WooCommerce
    • WooCommerce Admin
    • WooCommerce PayPal Checkout Gateway
