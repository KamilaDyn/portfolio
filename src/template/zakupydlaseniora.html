Zakupy Dla Seniora Wordpress (Shopping for Seniors)

Website in child wordpress theme, based on free Firm Graphy Child Theme. (link to theme)

ENG

Description

This website was created for postgraduate study mobile and web applications. It was created in team work (with 3 members). The goal of the website 'Zakupy Dla Seniora' (Shopping for Seniors), is to help make shopping for seniors who cannot do it by themselves during pandemia covid-19.

Website has:

  1. Front pages

    • Section about us,
    • Section with information: links to shopping form, contact and description how the help look like.
  2. Page with form. It is for seniors who can fill it up and send. The data is sent to a database in mysql. Senior can put his name, address, and shopping list.
  3. Page with map where are all orders which were not finalized. Map with orders  can be seen by all users, but just logged users can see all data with name and address.

  4. Section for volunteers with subpages:

    • Registration page, who want to be a volunteer can register on the page but administrators accept the person.
    • My account: Volunteer can login into account and change his own password.  
    • My orders: Volunteer who has an account can see all orders which he took to accomplish.If a volunteer doesn't want to make an order can send it back to the database.
    • All orders. Page where a volunteer can see all orders in a table.
  6. 5. Contact page with contact form.
  7. 6. Forum. All logged users can use the forum to ask questions, exchange experiences. 

  8. Plugins which we used in project:

    • Contact Form 7
    • ReCaptcha v2 for Contact Form 7
    • Database for WPforms
    • File Upload Types
    • Forum Asgaros
    • Wordpress File Upload
PL

Opis

Projekt podsumowujący studia podyplomowe. Strona stworzona w pracy grupowej – trzyosobowej. Storna ma na celu pomóc seniorom z zakupami, którzy nie mogą zrobić ich sami w czasie pandemii covid-19.

Strona zawiera:

  1. Stornę główną:

    • Sekcja o nas.
    • Sekcja skrót informacjami z linkiem do formularza, danych kontaktowych i opisem jak wygląda pomoc
    • Sekcja informacjami z linkami do rejestracji dla wolontariusza lub formularza dla seniora
  2. Strona z formularzem, przeznaczona dla seniora Senior może wypełnić prosty formularz, wszystkie dane z formularza są wysyłane do bazy danych. Senior może podać swoje dane osobowe, dane zamieszkania oraz listę produktów.
  3. Strona mapa zleceń. Zlecenia są wyświetlane w zakładce przeglądaj zlecenia oraz na dołączonej mapie. Do danych osobowych ma dostęp tylko osoba zalogowana, która jest zaakceptowana prze administratora. Jeśli osoba nie jest zarejestrowana, na mapie zleceń jest wyświetlana lista zakupów danej osoby bez danych osobowych z przybliżonym adresem zamieszkania.

  4. Sekcja dla wolontariusza z podstronami:

    • Rejestracja: osoba która chce pomóc w zakupach może zostać wolontariuszem i zarejestrować się na stronie, administrator akceptuje konto. Po akceptacji wolontariusz może korzystać z konta.
    • Moje Konto: Wolontariusz ma możliwość logowania się na stronie, zmienić swoje hasło
    • Moje zlecenia: Wolontariusz ma możliwość przeglądać zlecenia, które przejął, zmienić status zlecenia, na zrealizowany lub zwrócić zlecenie do bazy danych.
    • Przegląd zleceń: Wolontariusz ma możliwość przeglądać zlecenia w bazie i je przejąć.
  5. Strona kontakt. Informacje kontaktowe, z formularzem kontaktowym
  6. Forum. Użytkownicy, wolontariusze mogą komunikować się między sobą, wymieniać doświadczenia przez forum.

  7. Pluginy wykorzystane w projekcie:

    • Contact Form 7
    • ReCaptcha v2 for Contact Form 7
    • Database for WPforms
    • File Upload Types
    • Forum Asgaros
    • Wordpress File Upload
